== Description ==
Enlight pricer tool is an awesome plugin that allows you to create wonderful tables which can be used on your WordPress site.  It has an awesome admin interface which allows you to select a variety of options and colors for your price table. The output of the table will be based on the bootstrap framework so it will not have any effect on your theme functionality. 

The plugin is totally based on Javascript and bootstrap framework so  there are negligible chances for any conflicts with other theme as well as plugins.

Ordered list:

1. Create colour full table
2. Table will totally be desined by admin UI
== Installation ==

= Admin Installer via search =
1. Visit the Add New plugin screen and search for "enlight pricer".
2. Click the "Install Now" button.
3. Activate the plugin.
4. Navigate to the "Enlight Pricer" Menu.

= Admin Installer via zip =
1. Visit the Add New plugin screen and click the "Upload Plugin" button.
2. Click the "Browse..." button and select zip file from your computer.
3. Click "Install Now" button.
4. Once done uploading, activate Enlight Pricer.

= Manual =
1. Upload the enlightpricer folder to the plugins directory in your WordPress installation.
2. Activate the plugin.
3. Navigate to the "Enlight Pricer" Menu.

That's it! Now you can easily start creating wonderfull and eyecatching Table for your site.
